# 📊 SQL Server Estudos

Este repositório contém meus estudos e exercícios utilizando **SQL Server**, com foco em aprender e praticar consultas SQL e manipulação de dados em bancos relacionais.

---

## 📁 Estrutura do Projeto

```
sql-server-estudos/
│
├── README.md
├── queries-basicas.sql
└── exercicios-where.sql
```

### 📄 Arquivos

**queries-basicas.sql**
Contém consultas SQL básicas utilizadas para aprendizado, incluindo comandos como:

* `SELECT`
* `DISTINCT`
* consultas simples em tabelas

**exercicios-where.sql**
Arquivo com exercícios focados no uso do comando `WHERE`, utilizando:

* operadores lógicos (`AND`, `OR`)
* filtros de dados
* desafios práticos

---

## 🗄️ Banco de Dados Utilizado

Os exemplos foram realizados utilizando o banco de dados de exemplo **AdventureWorks**, muito utilizado para aprendizado em SQL Server.

## 💻 Tecnologias

* SQL

* SQL Server Management Studio

Registrar minha evolução no aprendizado de SQL e praticar consultas em banco de dados.
